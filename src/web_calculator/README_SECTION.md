# web_calculator package
- `app.py`: entrypoint for the desktop app; wires UI, controllers, and services.
- `__init__.py`: package marker.
- Subpackages:
  - `config/`: placeholder for app configuration files.
  - `core/`: domain logic (pricing, models, services, payload builders).
  - `data/`: bundled JSON/ico assets for catalog, packages, services, supplier, PDF content.
  - `ui/`: Tk UI layouts, controllers, and components.
  - `utils/`: helpers for PDF, QR, variable symbol, etc.
